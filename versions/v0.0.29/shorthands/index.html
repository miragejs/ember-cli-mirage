Shorthand reference

A shorthand is a simple way to write a route handler for common API scenarios. Hereâ€™s a reference of the shorthands, and the code they represent.

Shorthands use default status codes, based on the HTTP verb:

  • GET is 200
  • PUT is 204
  • POST is 201
  • DEL is 204

PUT and POST change to 200 if there is a response body.
